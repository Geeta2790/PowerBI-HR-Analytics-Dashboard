# PowerBI-HR-Analytics-Dashboard
End-to-end HR Analytics Power BI Dashboard with DAX
# project overview 
This project focuses on building an end-to-end HR Analytics dashboard using Power BI to track key HR metrics such as headcount, attrition, hiring trends, and workforce insights.
#  Key KPIs
- Total Headcount
- Monthly Joinings & Exits
- Attrition Rate
- Department-wise Headcount
